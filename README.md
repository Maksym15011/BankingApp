# System Bankowy Online

Projekt stworzony w ramach zaliczenia przedmiotów Banking oraz Web Development.

## Opis projektu

System umożliwia:

- Rejestrację użytkowników
- Logowanie użytkowników
- Wyświetlanie salda konta
- Wykonywanie przelewów między użytkownikami
- Przeglądanie historii transakcji

## Technologie

### Backend

- Node.js
- Express.js
- JWT Authentication
- bcrypt

### Baza danych

- Microsoft SQL Server

### Frontend

- HTML
- CSS
- JavaScript

## Struktura projektu

```text
BankingApp
│
├── backend
├── frontend
└── database
```

## Funkcjonalności

### Użytkownicy

- Rejestracja
- Logowanie
- Podgląd salda

### Operacje bankowe

- Przelewy środków
- Historia transakcji
